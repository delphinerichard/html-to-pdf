## Project setup

This project needs to have `chromium` installed. It must be in `usr/bin/chromium`

To install all the Node dependencies:
```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
